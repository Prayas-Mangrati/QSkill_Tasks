# QSkills Internship Tasks

This repository contains the projects completed as part of my internship assignment. The objective was to strengthen frontend development skills using **HTML, CSS, and JavaScript** by building practical web applications and responsive user interfaces.

## Projects Included

### 1. Login Form with Validation

A modern login system built using HTML, CSS, and JavaScript.

#### Features

* Form Validation
* Custom Alert Modal
* Password Visibility Toggle
* Forgot Password Functionality
* Local Storage Integration
* Responsive Design

**Folder:** `LoginForm`

---

### 2. Razorpay Landing Page Clone

A responsive Razorpay-inspired landing page built from scratch.

#### Features

* Responsive Navigation Bar
* Mobile Hamburger Menu
* Country Selector Dropdown
* Animated Brand Marquee
* Product Showcase Section
* Security & Compliance Section
* Call-to-Action Section
* Modern Footer Design
* Scroll Reveal Animations

**Folder:** `Razorpay_Clone`

---

### 3. Spotify Song Downloader

A Spotify-themed web application that fetches song details and generates download links using RapidAPI.

#### Features

* Spotify-Inspired User Interface
* Spotify Track URL Validation
* Dynamic Song Information
* Album Artwork Display
* Song Download Link Generation
* Loading Animation
* API Integration with RapidAPI
* Error Handling

**Folder:** `SpotifySongDownloader`
