# Razorpay Clone

A modern and responsive Razorpay-inspired landing page built using HTML, CSS, and JavaScript. This project recreates the look and feel of a professional fintech website while focusing on responsive design, interactive UI components, and smooth user experience.

## Overview

This project is a frontend clone inspired by the Razorpay website. It showcases modern web development practices including responsive layouts, animations, custom dropdowns, mobile navigation, and interactive sections.

## Features

### Responsive Navigation

* Desktop and mobile navigation menus
* Hamburger menu for smaller screens
* Auto-closing mobile menu after navigation

### Custom Country Selector

* Flag-based country selection
* Interactive dropdown menu
* Click-outside detection for closing dropdown

### Hero Section

* Razorpay-inspired landing page design
* Responsive layout
* Call-to-action buttons

### Brand Marquee

* Infinite scrolling brand logos
* Smooth animation
* Pause on hover functionality

### Finance Platform Section

* Feature showcase
* Interactive hover effects
* Clean and modern layout

### Products Section

* Product cards with icons
* Hover animations
* Statistics dashboard

### Security & Trust Section

* Security-focused feature cards
* Modern card-based design

### Call To Action Section

* Conversion-focused design
* Interactive buttons

### Footer

* Multi-column layout
* Product and company links
* Responsive design

### Animations

* Scroll reveal animations
* Smooth transitions
* Hover interactions
* Brand marquee animation

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Font Awesome Icons
