# Login Form with Validation

A responsive login form built using HTML, CSS, and JavaScript featuring form validation, custom alerts, password visibility toggle, and password recovery functionality.

## Features

* User Login Form
* Username Validation
* Password Validation
* Custom Alert Popups
* Show / Hide Password
* Forgot Password Functionality
* Password Reset using Local Storage
* Responsive UI Design
* Modern Two-Panel Layout

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Local Storage API

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.

No additional installation or dependencies are required.

## Functionalities

### Login Validation

* Checks for empty fields.
* Validates minimum username length.
* Validates minimum password length.
* Displays custom success and error alerts.

### Password Visibility Toggle

* Users can show or hide their password using the eye icon.

### Forgot Password

* Users can reset their password through the Forgot Password modal.
* Passwords are stored using the browser's Local Storage.
* Updated passwords can be used immediately for login.
